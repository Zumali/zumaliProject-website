# Zumali Project Website

This repository contains the source code for the official **Zumali Project** landing page.

The website serves as the public face of Zumali, introducing our mission, vision, and the journey of building a software startup from the ground up.

---

## 🚀 Features

- Establish and showcase the Zumali brand identity.
- Responsive design optimized for desktop, tablet, and mobile devices.
- Present the company's mission and vision.
- Document the founder's journey as Zumali grows.
- Provide a foundation for future website expansion.

---

## 🛠️ Built With

### Frontend
- HTML5
- CSS3
